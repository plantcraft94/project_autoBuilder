How to share your build

1. Export your build here: https://www.marcrobledo.com/savegame-editors/zelda-totk/

2. Upload an image of your build online via Imgur or any other platform, and copy the link(the link have to and with .png).

3. Go to the Upload tab and fill in all the information about your build, including the build's name, a link to the image of your build, the .cai file obtained from the savegame editor, and the description of your build (you can include details such as the list of required Zonai devices and the number of Zonaite it costs).

4. Click on the Upload button and wait for the upload to finish.

How to download a build

1. Go to the Home page, find the build and click on "Detail", it will take you to the detail page of the build you choose.

2. Click on the Download button, which will take you to a pixeldrain.com page.

3. Download the .cai file from the gofile.io page.

4. Import the build in the savegame editor at https://www.marcrobledo.com/savegame-editors/zelda-totk/

NOTE:

  • You should always backup your savegame before importing/exporting a build.
  • After uploading the build, you can't edit or delete it anymore because there is no account system, so please be careful.
  • The save editor's author change the file from .totkab to .cai so if you upload a build with a .totkab file, please reupload it with a .cai file because the .totkab files don't work

Have fun!
